# n5

A .NET REST API that returns Permissions services.


## Getting Started

### Install

```bash
docker compose up
```

```bash
 dotnet restore
 dotnet tool install --global dotnet-ef                 
 dotnet tool update --global dotnet-ef   
 dotnet ef         
 dotnet ef database update
```

```bash
dotnet run
```
